### 代码示例

```python
# 1. 用户提交内容
POST /api/upload
{
    "content": "今天我们来讲胃液的分泌...",
    "date": "2026-02-18"
}

# 2. AI解析
ai_result = await content_parser.parse(content)
# {
#     "book": "生理学",
#     "chapter_id": "physiology_ch06_03",
#     "chapter_title": "胃内消化",
#     "concepts": [
#         {"id": "..._胃液成分", "name": "胃液成分", "importance": "main"},
#         {"id": "..._消化调节", "name": "消化液分泌的调节", "importance": "secondary"}
#     ]
# }

# 3. 保存到数据库
db.add(DailyUpload(...))
db.add(Chapter(...))
for concept in ai_result.concepts:
    db.add(ConceptMastery(...))
db.commit()
```

## 🧠 核心算法: 间隔重复 (FSRS)

```
记忆保留率随时间下降
│
1.0 ┤────┐
    │    │＼
0.8 ┤    │  ＼    ← 第1次复习后，间隔1天
    │    │    ＼
0.6 ┤    │      ＼
    │    │        ＼
0.4 ┤    │          ＼  ← 第2次复习后，间隔3天
    │    │            ＼
0.2 ┤    │              ＼
    │    │                ＼
0.0 ┼────┴──────────────────┴───→ 时间
    今天  1天后  3天后  7天后  14天后

算法逻辑:
if 答对:
    mastery_level += 1
    interval = [1, 3, 7, 14, 30][mastery_level]
    next_review = today + interval天
    
    if mastery_level >= 3:
        is_mastered = True

if 答错:
    mastery_level = max(0, mastery_level - 1)
    next_review = tomorrow
    is_mastered = False
```

## 🎯 三种练习模式的区别

| 模式 | 题目来源 | 适用场景 |
|------|---------|---------|
| **practice** (正常练习) | AI实时生成新题 | 检验学习效果，发现薄弱点 |
| **wrong_answer_review** (错题复习) | 错题本中未掌握的题 | 针对性复习，消灭错题 |
| **repeat** (重复练习) | 历史做过的题目 | 巩固记忆，防止遗忘 |

## 🔑 关键设计决策

### 1. 双轨存储
```
日期轨道: DailyUpload → 保留原始记录，不可修改
章节轨道: Chapter + ConceptMastery → 结构化知识，可更新
```

### 2. 错题独立性
```
错题本保存完整题目内容
即使原题删除，错题本仍可复习
```

### 3. 掌握度分离
```
ConceptMastery.retention → 整体记忆度 (0-1)
WrongAnswer.mastery_level → 该题的掌握度 (0-5)
```

### 4. 智能复习推荐
```
今日任务 = {
    复习: 下次复习时间 <= 今天的知识点,
    新学: 刚上传的章节,
    错题: 未掌握且今天该复习的错题
}
```

---

这个系统本质上是一个**闭环学习系统**:

```
上传 → 学习 → 测试 → 发现错题 → 复习 → 掌握 → 继续学习
  ▲                                              │
  └──────────────────────────────────────────────┘
```

每个环节都有数据记录，AI根据数据智能推荐下一步学习内容。
