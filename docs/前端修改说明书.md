## 一、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端框架 | FastAPI (Python 3.12) | 路由 + Jinja2 模板渲染 |
| 数据库 | SQLite + SQLAlchemy ORM | 数据文件: `data/learning.db` |
| CSS | Tailwind CSS (CDN) | `https://cdn.tailwindcss.com` |
| 图表 | Chart.js 4.4.7 (CDN) | 用于趋势图、分布图 |
| JS | 原生 JavaScript | 无框架，所有 JS 内联在模板中 |
| 模板引擎 | Jinja2 | 继承 `base.html`，block: title/content/extra_css/extra_js |
| 静态资源 | 无 | `static/` 目录为空，全部走 CDN 或内联 |

## 三、基础模板 `base.html`

```html
<!-- 导航栏：紫色渐变 gradient-bg -->
<nav> 3个链接：仪表盘(/) | 学习轨迹(/learning-tracking) | 错题本(/wrong-answers) </nav>

<!-- 主内容区 -->
<main class="max-w-7xl mx-auto px-4 py-8">
    {% block content %}{% endblock %}
</main>

<!-- 页脚 -->
<footer>True Learning System - AI驱动的学习助手</footer>
```

所有页面通过 `{% extends "base.html" %}` 继承，使用以下 block：
- `{% block title %}` — 页面标题
- `{% block extra_css %}` — 额外样式
- `{% block content %}` — 主内容
- `{% block extra_js %}` — 额外脚本

## 七、前端修改注意事项

### 7.1 开发环境

```bash
# 启动开发服务器
cd true-learning-system
pip install -r requirements.txt
python main.py
# 访问 http://localhost:8000
```

### 7.2 修改规则

1. **不要动后端文件**: `routers/`、`services/`、`models.py`、`schemas.py` 不要修改
2. **模板继承**: 所有新页面必须 `{% extends "base.html" %}`
3. **API 契约**: 所有 API 的请求/响应格式已固定，前端只能适配，不能要求后端改接口
4. **CDN 依赖**: Tailwind 和 Chart.js 通过 CDN 加载，不要引入本地构建工具
5. **编码**: 所有文件 UTF-8，中文界面

### 7.3 测试方法

修改后需验证：
- 各页面能正常打开（无白屏）
- API 调用正常（浏览器 F12 Network 无红色请求）
- 数据展示正确（对比修改前后）
- 键盘快捷键仍可用（测验页面 1-5 选项、Q/W/E 自信度）

## 八、数据库模型（只读参考）

```
DailyUpload (上传记录) → Chapter (章节) ← ConceptMastery (知识点)
                                                    │
                              ┌──────────────────────┼──────────┐
                              ▼                      ▼          ▼
                        WrongAnswer          TestRecord    QuizSession
                         (错题本)             (测试记录)     (测验会话)

LearningSession (学习会话) → QuestionRecord (答题记录)
WrongAnswerV2 (错题本V2，含SM-2间隔重复)
FusionQuestion (融合题)
```

核心表：
- `chapters` — 章节信息
- `concept_mastery` — 知识点掌握度
- `wrong_answers_v2` — 错题本（V2，主力使用）
- `learning_sessions` — 学习会话
- `question_records` — 答题记录
- `quiz_sessions` — 测验会话
