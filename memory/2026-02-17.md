### 2. 错题本系统完整实现 ✅

#### 核心功能
- **固定10道题**: QuizSession模型，total_questions=10
- **错题记录**: WrongAnswer模型，自动保存答错题目
- **三种练习模式**:
  - `practice`: 正常练习，生成新题目
  - `wrong_answer_review`: 错题复习，只出未掌握错题
  - `repeat`: 重复练习，复习已做过的题目

#### 数据库模型
```python
WrongAnswer:
  - concept_id          # 知识点ID
  - question            # 题目内容
  - options             # 选项 (JSON)
  - correct_answer      # 正确答案
  - user_answer         # 用户答案
  - error_type          # 错误类型
  - review_count        # 复习次数
  - mastery_level       # 掌握程度0-5
  - is_mastered         # 是否已掌握
  - next_review         # 下次复习时间

QuizSession:
  - session_type        # 'practice'/'wrong_answer_review'/'repeat'
  - questions           # 10道题的完整数据
  - answers             # 答题记录
  - score               # 得分
```

#### 智能复习算法
```
答对 → mastery_level +1
     → 下次复习时间延长 [1, 3, 7, 14, 30]天
     
答错 → mastery_level -1
     → 明天继续复习
     
mastery_level >= 3 → 标记为已掌握
```
